# Vim Installer

This will install a vim editor with configurations for PHP development.
 
---

## Installation

```shell
$ bash installer.sh
```
---

## What will be installed

- Git
- Vim
- GVim
- PHP
- Composer
- Ag
- CTags
- Drush
- Drupal Coder
- PhpCs
- CMake
- Jetbrains Font
- Vundle
- Python3
